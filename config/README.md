# config — 配置文件目录

本目录存放项目运行与算法相关的**配置参数**，统一使用 YAML 格式，便于修改和版本管理。

---

## 文件说明

| 文件 | 说明 |
|------|------|
| **default.yaml** | 默认全局配置，包含导航、检测、除草策略等所有可调参数。 |

---

## default.yaml 结构概览

- **navigation**  
  导航相关：垄距先验、垄线角度阈值、避障安全距离、轨迹安全距离、路径速度与步长等。

- **detection**  
  检测相关：杂草类别数、根深/生长阶段数、置信度与 NMS 阈值、输入尺寸等。

- **weed_control**  
  除草策略表：按杂草类型（0、1、2…）配置「功率(W)、时间(ms)、深度(cm)」及根深等级；另含幼苗期功率缩放 `seedling_power_scale`。

修改后无需改代码即可生效（脚本/程序需从该文件读取配置）。
